5. **Compile**:
   ```bash
   g++ main.cpp -o main.exe
   ```

6. **Run**:
   ```bash
   ./main.exe
   ```


4. **Write Code**:
   - Using `nano`, `vim`, or create in an external editor and move to Termux.

   ```bash
   clang++ main.cpp -o main
   ```

6. **Run**:
   ```bash
   ./main
   ```
